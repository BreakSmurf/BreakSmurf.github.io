Анализатор логов (Blue Team)

Скрипт для анализа SSH auth-логов и веб access-логов (Apache/Nginx combined format): находит brute-force атаки на SSH, SQL-инъекции, XSS, path traversal и автоматическое сканирование директорий — и выдаёт отчёт с рекомендациями по каждой находке.

Логика работы

  • Регулярки парсят стандартный формат строк sshd (Failed password / Accepted password) и Apache/Nginx combined log format.
  • Для SSH — считаются неудачные попытки на IP; если их ≥5, это находка; если после этого с того же IP был успешный вход — severity повышается до high (возможный компромисс).
  • Путь запроса в веб-логе декодируется (urllib.parse.unquote) и проверяется набором сигнатур: SQLi, XSS, path traversal — чтобы не пропустить обфусцированные (%20, %3C) попытки атак.
  • Если IP сделал ≥10 запросов и больше 60% из них — 404, это помечается как автоматическое сканирование директорий (dirbuster/gobuster-подобное поведение).

Демонстрация работы

Запуск на встроенных тестовых логах (sample_logs/) — они синтетические, специально собраны для демонстрации, а не взяты с реального сервера. Вывод скрипта настоящий.

powershell
