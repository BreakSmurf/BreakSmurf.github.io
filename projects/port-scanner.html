Сканер портов и сервисов на Python

Учебный многопоточный TCP-сканер портов без внешних зависимостей: проверяет диапазон портов, пытается определить сервис и забрать баннер, экспортирует отчёт в JSON/CSV. Написан для того, чтобы понять, как устроено сканирование портов «изнутри», а не только пользоваться готовым nmap.

Логика работы

  • Хост резолвится в IP через socket.gethostbyname.
  • Каждый порт проверяется в отдельном потоке (ThreadPoolExecutor, до 200 воркеров по умолчанию) через socket.connect_ex — так проверка не блокирует остальные порты.
  • При успешном подключении сканер читает первые байты ответа (banner grabbing) — многие сервисы (FTP, SSH, SMTP) сами присылают приветственную строку.
  • Порт сопоставляется с известным сервисом по номеру (SSH, HTTP, RDP, MySQL и т.д.) из встроенного словаря популярных портов.
  • Результаты сортируются по номеру порта и по флагу сохраняются в JSON или CSV.
